作为一名码农，文本工具肯定是必不可少的。个人感觉工具这个东西，无所谓好坏，无所谓高端低端，一个够用，再一个顺手就足够了。之前记得刚入行两年的时候为了选一款顺手的文本工具， 主流的文本编辑器基本都尝试过：Notepad++、editplus,最后都没留下，近几年一直用的是一款国产自研的文本编辑器EverEdit。之所以后来一直用这个，说起来是因为一个很小的功能， 就是这个编辑器可以像浏览网页一样，按住 Ctrl 键，滑动鼠标滚轮可以放大缩小整个文本。这里多聊一句，作为开发者更应该多站在用户的角度去考虑产品，而不要只局限于软件开发。其他 文本工具绝不是不够强大，可能相反无论知名度还是功能可能都要强大很多。想想当时还是刚入行一年的小码农，为了解决问题、提升自己，经常无偿自己在公司加班，为了查资料肯定会浏览 大量的网络资料，时间一久，眼镜就会干涩，为了眼睛舒服点，就下意识的 Ctrl + 滚轮，就是这么个小习惯，那些强大的编辑器不支持（后来和同事交流，有的说支持这个，可能是版本问题 没有验证），后来慢慢没在用。

说了这么多，引出今天的主角 Visual Studio Code,号称微软的良心之作
