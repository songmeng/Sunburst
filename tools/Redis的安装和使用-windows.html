最近学习一个基于springboot的开源项目需要用到Redis，马上开始Window下Redis的安装

1.Redis的下载安装

1.1获取Window环境下Redis的安装包，访问github下Redis工程https://github.com/MSOpenTech/redis

1.2点击release,进入稳定版本列表，选择最新版本下载Redis-x64-3.0.504.msi

1.3下载完毕，找到Redis-x64-3.0.504.msi开始安装

一路选择next,配置选择默认配置就可以，稍等片刻安装完成

2.Redis简单配置

2.1安装完毕后，需要先做一些设定工作，以便服务启动后能正常运行。使用文本编辑器，使用vscode打开Redis服务配置文件。注意：不要找错了，通常为redis.windows-service.conf，而不是redis.windows.conf。后者是以非系统服务方式启动程序使用的配置文件

2.2找到含有requirepass字样的地方，追加一行，输入requirepass 12345。这是访问Redis时所需的密码，一般测试情况下可以不用设定密码。不过，即使是作为本地访问，也建议设定一个密码。亲测这种方式无效，可能是版本问题

2.3可能是版本问题按上面配置后，在使用时报错：(error) ERR Client sent AUTH, but no password is set

解决方法为，在控制台设置密码

    127.0.0.1:6379> config set requirepass 12345
    OK

出现OK了，说明设置成功

3.Redis简单测试使用

3.1首先确保redis服务已启动，点击“开始”>右击“计算机”>选择“管理”。在左侧栏中依次找到并点击“计算机管理（本地）”>服务和应用程序>服务。再在右侧找到Redis名称的服务，查看启动情况。如未启动，则手动启动之。正常情况下，服务应该正常启动并运行了。

3.2打开redis安装目录，运行redis-cli.exe，或cmd执行也可以

3.3先验证是否安装配置成功，命令行执行"anth 123456",123456是redis密码,返回ok说明配置运行成功

    127.0.0.1:6379> anth 123456
    OK

3.4Redis简单读写测试

    127.0.0.1:6379> set testKey "testData!"
    OK
    127.0.0.1:6379> get testKey
    "testData!"
    127.0.0.1:6379>

能够成功存取数据，说明运行成功，可以码代码了！
