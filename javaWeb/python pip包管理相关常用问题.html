pip模块安装相关问题

1.Python的pip安装模块目录

  • 再次输入pip install xxxx就会显示此包已经安装的路径了
  • 其实所有通过System pip安装的包都在此目录下，如：
        C:\Users\Admin>pip install numpy
        Requirement already satisfied: numpy in c:\users\admin\appdata\local\programs\python\python38-32\lib\site-packages (1.24.1)

即：c:\users\admin\appdata\local\programs\python\python38-32\lib\site-packages

2.如何将模块只安装在项目的venv中

  • 手动操作安装，Intellij Idea右下角点击，选择对应的python interpreter环境，手动搜索安装
  • pip命令行安装，注意只安装在所在项目venv中的话，命令行的区别
        python -m pip install  xxxx

注意前缀： python -m

3.如何安装xxx.whl文件

  • 安装至公共System库:cmd进入.whl文件目录，直接命令行运行安装
  • 进一步，只安装在项目的venv中，注意前缀。前提：将.whl文件放置venv文件夹内
        pip install xxxx.whl
        python -m pip install xxxx.whl

4.直接源代码安装，github直接下载项目源码

  • 安装至公共System库:cmd进入git源码文件目录，直接命令行运行安装
  • 进一步，只安装在项目的venv中，注意前缀。前提：将源码文件放置在venv文件夹内
        python setup.py install
        python -m setup.py install
